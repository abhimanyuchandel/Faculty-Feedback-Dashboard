# Deployment Examples

The primary supported deployment path for this repository is Vercel, documented in [../docs/DEPLOYMENT.md](../docs/DEPLOYMENT.md).

This folder keeps alternate deployment experiments separate from the main operational path:

- [Netlify](./netlify/README.md)
- [Cloud Run](./cloud-run/README.md)
- [Cloudflare/OpenNext](./cloudflare/README.md)

These guides are intentionally secondary so future contributors can find the production path first.
